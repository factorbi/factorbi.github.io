## Brand
- Product name: **Vansync** (never "Bipost" — rebranded)
- Executable: `Vansync.exe` (old: `biPost.exe`)
- S3 bucket prefix: `vansync` (old: `bipostdata`)
- Tagline: Extract, transform and load data from legacy on-premises databases to AWS Aurora MySQL — unlocking modern BI tools like Amazon QuickSight and Google Looker Studio for databases that lack native cloud connectivity.
- Company: **Factor BI**
- Supported source DBs: FirebirdSQL, Sybase SQL Anywhere, dBase/DBF, Visual FoxPro, SQL Server, MySQL

### Leave intact (do not rename)
- AWS system database names: `bipost_system`, `bipost_sync_info`, `bipost_sync_table`
- Image filenames (e.g. `img/bipost7.png`)
