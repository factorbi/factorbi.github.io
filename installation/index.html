Before you install¶

If you haven't linked your RDS Instance to Bipost API, please follow these steps: Link your AWS account.

Prerequisites¶

For most Windows 7 and up there is no need to install the prerequisites. Nevertheless if you face any trouble running biPost.exe, please try with the following.

Window XP & Vista¶

Download and install:

  • WindowsInstaller.
  • Prerequisites for Windows 7 and 8.

Windows 7 and 8¶

Download and install:

  • dotNetFx40
  • vcredist

Download & Install¶

  1. Get latest version here.

  2. Unzip biPost.zip to any folder on your Windows.

  3. Check if you can successfully launch biPost.exe

  4. Configure Bipost Sync
